# Screenshot inventory

The main `README.md` references the PNGs in this folder. Capture them on a freshly installed instance with at least one project configured and Claude signed in, then drop them in alongside this file.

| File | What to capture |
|---|---|
| `dashboard.png` | Main dashboard with 2-3 project cards visible. |
| `dashboard-empty.png` | The Welcome card a fresh install lands on (no projects yet). |
| `wizard.png` | Setup Wizard modal open at the CLIs section. |
| `terminal.png` | Project terminal page with the top bar (back / project name / tab strip / Preview / Files) and an active Claude Code prompt. |
| `preview.png` | Preview modal with a running ASP.NET or Vite app inside the iframe. |
| `manage.png` | Manage Projects page with the Add form and one or two project cards (showing preview-cmd textareas). |

Recommended dimensions: 1600px wide, PNG, sRGB. Crop to the workbench UI; don't include the OS chrome.
